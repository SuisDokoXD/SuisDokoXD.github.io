关于我

嗨，我是 SuisDokoXD 👋

厦门理工学院 PFA 战队 成员，目前在 RoboMaster 导航组埋头折腾哨兵机器人。

我在做什么

🤖 哨兵自主导航 (RM2026)
基于 Mid360 LiDAR 的 SLAM 导航开发，主线任务：让哨兵在赛场地形里不要再把斜坡当成墙。

✈️ 掌上飞机 (副业)
ESP32-C3 + IMU660RA(BMI270) 的掌上 ESP-Drone。MicroPython 路线已跑通 IMU 全链路（互补滤波 + WiFi 3D 可视化）。

技术栈

  • 机器人：ROS2 / Nav2 / FAST-LIO / PCL
  • 嵌入式：ESP32 / STM32 / MicroPython / C++
  • 算法：SLAM / 姿态解算 / 点云处理

为什么写博客

把踩过的坑、调过的参数、想过的问题留下来，给未来的自己看，也希望能帮到同样在折腾机器人的人。

联系方式：右下角找我，或者 GitHub Issues 留言。
