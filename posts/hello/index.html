终于把博客搭起来了

折腾了不到一小时，从零到上线。技术栈也没什么花哨的：

  • 生成器：Hexo 7.x
  • 主题：Butterfly (27k+ star，国产主题之光)
  • 部署：GitHub Pages
  • 域名：先用 suisdokoxd.github.io，后面可能换 .me

这个博客会写什么

主要是技术向，预计的几个方向：

🤖 哨兵导航相关

  • Mid360 LiDAR + FAST-LIO 调参经验
  • ROS2 Nav2 在 RM 赛场的实战
  • 斜坡地形误识别的排查全过程（这是当前最头疼的）
  • TF 树 / costmap / behavior tree 笔记

✈️ 嵌入式开发

  • ESP32-C3 + MicroPython 路线避坑
  • IMU 数据融合（互补滤波 / EKF / 卡尔曼）
  • 掌上飞机从 0 到能飞的全记录

🛠️ 工具链

  • Claude Code / Cursor 工作流
  • Linux 开发环境配置
  • 偶尔写点小工具的源码解读

为什么写博客

三个原因：

  1. 给未来的自己看。调试机器人时踩的每一个坑，过几个月就忘了，写下来比记在脑子里靠谱
  2. 强迫自己想清楚。能写明白才算真懂
  3. 认识更多折腾机器人的朋友。RM 圈子不大，希望能交到同好

关于这个博客

如果你也是 RM 选手、嵌入式爱好者、或者只是路过的同学，欢迎在 GitHub Issues 留言聊聊。
