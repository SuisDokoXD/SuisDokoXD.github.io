我在听

— 言って。

ヨルシカ / Yorushika

来自日本的双人音乐组合。n-buna 写词作曲编曲，suis 演唱。

第一次听他们是因为 「だから僕は音楽を辞めた」。后来意识到，他们的每张专辑都是一篇完整的小说，每首歌都是其中一页 —— 文学性 + 旋律性，能同时打到的乐队不多。

n-buna 笔下的引用谱系：太宰治、萩原朔太郎、ヘミングウェイ、夏目漱石……听他的歌像在读现代日本文学的二次创作。

专辑年表

🎒 だから僕は音楽を辞めた (2019)

第一张全长专辑。以”放弃音乐的青年写给恋人的最后一封信”为线索串起整张专辑。如果只能听一张，先听这张。

推荐曲：

  • だから僕は音楽を辞めた
  • 言って。
  • 八月、某、月明かり
  • パラノイア

✉️ エルマ (2019)

だから僕は音楽を辞めた 的姐妹篇。从恋人 Elma 的视角回信，两张专辑互文。

推荐曲：

  • 雨とカプチーノ
  • 海底、月明かり
  • 心に穴が空いた

📕 盗作 (2020)

以”盗作小说家的告解”为概念。比前两张更阴郁，编曲更厚。

推荐曲：

  • 春ひさぎ
  • 思想犯
  • 花に亡霊
  • 盗作

🌙 月光 (2023)

最新一张全长。回到清澈的旋律感，但叙事更克制。

推荐曲：

  • だから僕は音楽を辞めた → 春泥棒 → 月に吠える

单曲精选（不分专辑的最爱）

歌名 一句话推荐
花に亡霊 「夏は失せぬ」最直击的副歌
春泥棒 樱花飘落时一定要听
言って。 副歌爆发力封神
ただ君に晴れ 整张《负け犬にアンコールはいらない》的画面感全在这首里
思想犯 「ねぇ、響け、響け」
靴の花火 听完想坐夜行火车去陌生城市
老人と海 致敬海明威
雨とカプチーノ 雨天单曲循环

为什么这页不放播放器

因为：

  1. 网页音乐版权我搞不定（中日双边都麻烦）
  2. 自动播放会被浏览器拦
  3. 真要听就去 Spotify / Apple Music / 网易云搜「ヨルシカ」 —— 完整版加歌词，体验比博客嵌入好十倍

还在听的别的

  • 結束バンド (Kessoku Band) —— 这博客的视觉来源
  • Aimer
  • Eve
  • 米津玄師

推荐什么好听的可以来 GitHub Issues 跟我聊。
